#### Problem Statement
The puropose of this project is to build a basic machine learning model that can estimate the price of car using different characterstics such as car type, color,name, fuel type.By analyzing this features, the model learns patterns from the dataset and attempts to predict the price of car using linear regression approach.

#### Dataset Description
The dataset used in this project was manually created for learning purposes. It contains information about different cars including their type, color, name, and fuel type. The target variable used for prediction is the car price.

## Model Used
A Linear Regression model from the Scikit-learn library was used to train the dataset and predict car prices. Before training the model, categorical features were converted into numerical values.

## Results
The performance of the model was evaluated using Mean Absolute Error (MAE) and R² score. These metrics help measure how accurately the model predicts the target values.

## Conclusion
This project demonstrates the basic machine learning workflow including dataset preparation, data exploration, visualization, model training, feature experimentation, and overfitting analysis using Python libraries such as Pandas and Scikit-learn.
